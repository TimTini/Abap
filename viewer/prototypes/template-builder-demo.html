Template Builder Demo

Standalone prototype for building template config visually. This page is isolated from the main viewer runtime. It does not patch the stable app and only writes to its own demo state.

Use this to shape FORM or other templates visually first, then port JSON into the real viewer later.

Live Preview

Primary editing surface. Click or drag directly on preview cells to select the range you want to edit.

Selection A1

Inspector

Edit the current range and anchor cell without typing raw range keys.

Template options

These map to the same option block shape used by the real viewer config.

Generated JSON

Editable. You can paste a full config object or a single template object here.

How to use

  1. Select a template key or create one from a starter preset.
  2. Drag or click directly on Live Preview to choose a range.
  3. Edit text and style in the inspector, then apply.
  4. Check the result, then copy or download the generated JSON.
  5. If the output looks right, port the JSON into the real viewer later.
